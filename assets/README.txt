PROJECT SCREENSHOTS — yahan apni app ki screenshots daalo
===========================================================

Niche diye gaye EXACT naam se images is folder mein rakho.
Format: PNG ya JPG. Best size: ~800x500 px (landscape).

  chattoken.png       -> ChatToken app screenshot
  cmtmeet.png         -> CMT Meet app screenshot
  boonect.png         -> Boonect app screenshot
  coolingmatrix.png   -> Cooling Matrix admin panel screenshot
  parlour.png         -> Parlour Management desktop app screenshot

Agar koi image nahi dali, to us card par screenshot apne aap
hide ho jayega (koi broken icon nahi dikhega).

RESUME:
  Apni resume PDF ko ROOT folder (index.html ke saath) mein
  is naam se rakho:  Chanda_Kumari_Resume.pdf

Images/PDF daalne ke baad terminal mein:
  git add -A
  git commit -m "Add screenshots and resume"
  git push
